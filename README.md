# hauntpet/theme

## Documenation
### Build
```php
Repository::add($key, [
    // metadata
], [
    // options
]);
```

### Options
Options can be added as the 3rd parameter of the `add` function.

#### Menus
By default there is always a `primary` menu.
```php
'menus' => [
    'secondary' => 'Secondary Menu',
]
```
